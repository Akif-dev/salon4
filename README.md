# Salon Website Project

## Overview
This is the documentation for the Salon Website project, developed to provide users with an efficient and user-friendly platform for salon services.

## Features
- User Registration and Login
- Appointment Scheduling
- Service Listings
- User Reviews and Ratings
- Admin Dashboard for Management

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Akif-dev/salon4.git
   ```
2. Navigate to the project directory:
   ```bash
   cd salon4
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
